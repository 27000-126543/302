## 1. 产品概述

3D智慧粮库综合运营与应急调度可视化平台，面向粮库管理人员和上级监管部门，通过3D可视化技术实现粮库全场景实时监控、智能出入库调度、粮情预警、检化验管理、虫害熏蒸审批、设备运维及权限管控，解决传统粮库管理中信息孤岛、响应滞后、调度粗放等痛点。

- 目标用户：粮库操作员、仓储部长、粮库主任、上级粮管部门
- 核心价值：全场景3D可视化+智能调度+多级审批闭环，实现粮库运营"看得见、管得住、调得快"

## 2. 核心功能

### 2.1 用户角色

### 2.2 功能模块

1. **3D粮库全景总览页**：3D场景展示全部仓房及设施，实时粮情数据叠加，点击交互查看详情
2. **智能出入库调度页**：入库最优分配+绿色路径动画，出库订单匹配+蓝色流动箭头，流量自动调节
3. **粮情预警与通风控制页**：粮温超限红色预警+自动通风启动+3D风机气流动画+预警推送
4. **检化验管理页**：批次关联、超标红色标签、自动生成整理工单、三级审批流程
5. **虫害熏蒸管理页**：虫害超阈值橙色闪烁、自动生成熏蒸方案、三级审批、3D紫色禁区
6. **设备运维管理页**：累计运行超阈值橙色标记、自动生成检修工单、备件通知
7. **权限与日志页**：四级权限管理、人脸识别登录日志、操作审计

## 3. 核心流程

### 3.1 粮温预警与通风流程
粮情传感器检测粮温 → 粮温>30℃ → 仓房3D模型变红 → 自动启动通风系统 → 3D风机旋转+气流动画 → 右上角推送预警通知 → 粮温恢复正常 → 通风自动停止

### 3.2 智能入库流程
粮食到库 → 检化验 → 系统根据品种和质量评分 → 匹配最优仓房（优质稻→准低温仓/普通稻→常温仓） → 有匹配仓房 → 绿色输送线动画指示入仓路径 → 更新库存；无匹配仓房 → 自动生成扩容申请工单 → 通知粮库主任审批

### 3.3 智能出库流程
出库订单下达 → 系统匹配最优仓房 → 规划输送路线 → 蓝色流动箭头动画 → 出库流量自动调节匹配装车速度 → 等待超15分钟 → 自动催办通知

### 3.4 检化验超标流程
检化验完成 → 水分/杂质超标 → 红色标签标记 → 自动生成整理工单（烘干/清理） → 质检员审批 → 仓储部长审批 → 分管领导审批 → 工单执行

### 3.5 虫害熏蒸流程
虫害密度检测 → 超阈值 → 区域3D模型闪烁橙色 → 自动生成熏蒸方案 → 安全员审批 → 仓储部长审批 → 粮库主任审批 → 3D显示紫色半透明禁区球体

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深蓝色(#0a1628)背景+科技蓝(#00d4ff)强调色+工业橙(#ff8c00)警告色+安全绿(#00ff88)正常色
- 按钮风格：圆角微光按钮，hover时发光效果
- 字体：标题使用Orbitron科技感字体，正文使用Noto Sans SC
- 布局风格：左侧3D主视区(70%)+右侧信息面板(30%)，顶部导航栏
- 图标风格：线性科技风图标（lucide-react）
- 整体风格：深色科技大屏风格，工业物联网仪表盘质感

### 4.3 响应式
- 桌面优先设计，最小支持1920x1080分辨率
- 大屏(>2560px)自动扩展3D视区
- 中屏(1024-1920px)信息面板可折叠
- 不支持移动端（工业大屏场景）

### 4.4 3D场景指导
- 环境/HDRI：深蓝色夜空工业风，微弱地面雾气
- 灯光：冷白色环境光+暖黄色点光源模拟仓区照明+蓝色体积光强调科技感
- 相机：透视相机，默认俯视45°，支持鼠标旋转/缩放/平移，点击仓房时相机平滑飞入
- 构图：平房仓6栋横排居中，立筒仓4组右侧，烘干车间左后，输送栈桥连接各仓，检化验中心前方，调度中心前方主位
- 交互：鼠标悬停高亮+信息面板，点击飞入+详情，右键旋转场景
- 后处理：Bloom发光效果+环境遮蔽+景深
- 性能预算：目标60fps，LOD自动切换，粒子数控制在2000以内
